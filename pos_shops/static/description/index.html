POS Multi Shop

POS Multi Shop is a module to manage multiple stores or shops in Point Of Sale. It allows the POS user to create multiple shops and assign these shops/stores to multiple POS. By default in POS there is no option such that POS user will see only selected product in different POS Session. This module allows the POS user to create multiple store/shop and only that product will be available in POS Session that are selected in the associated shop.

For ex: If a seller has 3 different shops, 1st for fruits and vegetables , 2nd for Men's or Women's clothing and Accessories , 3rd for grocery store. In such a situation, 3 different shops are created with associated product list and assign these shops to the corresponding Point Of Sale, So that only those product will be available in POS Session that are selected in associated shop.

This module also provide the options to show promotions and these promotional description will be shown on the xml receipt(POSBOX). for ex: If seller want to show 'Christmas Discount Offer', then seller can create promotion for Christmas Discount Offer and this promotion will be shown in xml receipt (POSBOX)

Features
  • This module allows the seller to create multiple shops/stores in POS.
  • Seller can select the products for each shop and only these products will be shown in the corresponding POS Session.
  • Shop image/logo will be shown in xml receipt(POSBOX).
  • It also provides the option to add promotion and these promotional description will be shown on the xml receipt(POSBOX).
